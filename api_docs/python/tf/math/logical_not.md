# tf.math.logical_not

### Aliases:

* `tf.Variable.__invert__`
* `tf.contrib.eager.Variable.__invert__`

``` python
tf.math.logical_not(
    x,
    name=None
)
```

Returns the truth value of NOT x element-wise.

#### Args:

* <b>`x`</b>: A `Tensor` of type `bool`.
* <b>`name`</b>: A name for the operation (optional).


#### Returns:

A `Tensor` of type `bool`.
