# tf.keras.backend.update_sub

``` python
tf.keras.backend.update_sub(
    x,
    decrement
)
```

Update the value of `x` by subtracting `decrement`.

#### Arguments:

* <b>`x`</b>: A Variable.
* <b>`decrement`</b>: A tensor of same shape as `x`.


#### Returns:

The variable `x` updated.
