# tf.keras.activations.softplus

``` python
tf.keras.activations.softplus(x)
```

Softplus activation function.

#### Arguments:

* <b>`x`</b>: Input tensor.


#### Returns:

The softplus activation: `log(exp(x) + 1)`.
