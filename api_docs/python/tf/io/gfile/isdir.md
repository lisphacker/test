# tf.io.gfile.isdir

``` python
tf.io.gfile.isdir(path)
```

Returns whether the path is a directory or not.

#### Args:

* <b>`path`</b>: string, path to a potential directory


#### Returns:

True, if the path is a directory; False otherwise
