# tf.gfile.DeleteRecursively

``` python
tf.gfile.DeleteRecursively(dirname)
```

Deletes everything under dirname recursively.

#### Args:

* <b>`dirname`</b>: string, a path to a directory


#### Raises:

* <b>`errors.OpError`</b>: If the operation fails.
