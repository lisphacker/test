# tf.io.gfile.rmtree

``` python
tf.io.gfile.rmtree(path)
```

Deletes everything under path recursively.

#### Args:

* <b>`path`</b>: string, a path


#### Raises:

* <b>`errors.OpError`</b>: If the operation fails.
