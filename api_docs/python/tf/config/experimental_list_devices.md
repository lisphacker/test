# tf.config.experimental_list_devices

### Aliases:

* `tf.config.experimental_list_devices`
* `tf.contrib.eager.list_devices`

``` python
tf.config.experimental_list_devices()
```

List the names of the available devices.

#### Returns:

Names of the available devices, as a `list`.
